## Binary Format
- Object Name (String):
  - String Length (Int32)
  - String Content (UTF-8 or UTF-16-LE)
  - Terminator (UInt8 or UInt16)
- Function Name (String):
  - String Length (Int32)
  - String Content (UTF-8 or UTF-16-LE)
  - Terminator (UInt8 or UInt16)

## Example
```
[Object Name]
0A 00 00 00  Length (10 = 9 chars + 1 terminator)
50 6C 61 79 65 72 43 68 61 72  "PlayerChar" in UTF-8
00           Terminator

[Function Name]
0C 00 00 00  Length (12 = 11 chars + 1 terminator)
4F 6E 44 61 6D 61 67 65 54 61 6B 65  "OnDamageTake" in UTF-8
00           Terminator
```
